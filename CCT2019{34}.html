01 Extracción Inicial

// USB PCAP → ZIP → pcap_chal.pcap

El archivo inicial es una captura USB (pcap2_*.pcapng) que contiene transferencia de un ZIP. Al extraer el ZIP obtenemos el pcap real del desafío.

bash — extraer pcap del USB
tshark -r pcap2_*.pcapng -Y "usb.transfer_type==0x03" -T fields -e usb.capdata | ...
unzip extracted.zip
# → pcap_chal.pcap (4,588 paquetes)
💡
El pcap contiene conversaciones ICMP ocultas, tráfico HTTP a fotoforensics.com, y un stream TCP cifrado con cryptcat en el puerto 4444.

02 Task 1 — PCAP (IRC Bot)

// Conversación ICMP oculta

Los paquetes ICMP contienen una conversación que revela el uso de cryptcat con key BER5348833 en el puerto 4444.

datos clave del ICMP
# Key de cryptcat (Twofish): BER5348833
# Puerto: 4444

// Descifrado del stream TCP

bash — extraer y descifrar
tshark -r pcap_chal.pcap -qz follow,tcp,raw,49 | xxd -r -p > stream49.bin
cryptcat -k BER5348833 -l -p 9999 < stream49.bin
# Resultado: ELF 64-bit — IRC Bot

// Análisis del IRC Bot

El binario ELF es un bot IRC que se conecta al canal #flag y envía la flag codificada con ROT13 variante + string reverse.

output — flag decodificada

03 Task 2 — Reverse Engineering (.NET)

// Archivo: re3_*.exe (Dotfuscator)

PE32 .NET ofuscado con Dotfuscator. GUI con 4 scrollbars que deben satisfacer ecuaciones simultáneas.

ecuaciones del validador goodBoy()
a + b + c + d = 711
a × b × c × d = 711,000,000
a > b > c > d

# Solución única:
a = 316, b = 150, c = 125, d = 120

// Generación de la key

lógica IL desensamblada
# Constructor: field_c = 177, byteA = [32 bytes estáticos]
# goodBoy(): byteA[i] XOR (field_c XOR b)
#          = byteA[i] XOR (177 XOR 150)
#          = byteA[i] XOR 39
# Resultado → hex string de 32 caracteres
output — 32-char hex key
31C02DCFDE2FCF727016E2A7054B6DA5

// 4c — Run-Length Encoding

Secuencia de dígitos 1-6. Cada dígito = cantidad de bits consecutivos alternando 0/1 (empezando con 0).

python — decodificar RLE binario
binary = []
current_bit = 0
for ch in crypto1c:
    binary.extend([current_bit] * int(ch))
    current_bit = 1 - current_bit
