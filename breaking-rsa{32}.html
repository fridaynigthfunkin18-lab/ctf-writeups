03 Análisis Criptográfico — Extracción de n

Las claves públicas SSH en formato OpenSSH están codificadas en base64 con estructura TLV (Type-Length-Value). Se extraen e y n decodificando la clave:

python3 — extraer_n.py
import base64, struct

pubkey = "AAAAB3NzaC1yc2EAAAADAQABAAACAQDrZh8oe8Q8..."
key_bytes = base64.b64decode(pubkey)
parts = []
idx = 0
while idx < len(key_bytes):
    length = struct.unpack('>I', key_bytes[idx:idx+4])[0]
    idx += 4
    parts.append(key_bytes[idx:idx+length])
    idx += length

e = int.from_bytes(parts[1], 'big')
n = int.from_bytes(parts[2], 'big')

print(f"e = {e}")
print(f"bits de n = {n.bit_length()}")
print(f"últimos 10 dígitos de n = {str(n)[-10:]}")

04 Factorización con el Algoritmo de Fermat

// Fundamento teórico

Si p ≈ q, entonces n = p·q ≈ a² donde a = √n. El algoritmo busca a tal que a² - n = b² (cuadrado perfecto), y entonces:

matemáticas
p = a + b
q = a - b
# Con primos muy cercanos, converge en pocas iteraciones
# Un módulo de 4096 bits se factoriza en segundos

// Implementación

python3 — factorizar.py
import math

def factorize(n):
    a = math.isqrt(n)
    while True:
        a += 1
        bsq = a * a - n
        b = math.isqrt(bsq)
        if b * b == bsq:
            return a + b, a - b

n = 960343778775549488...1225222383  # n completo
p, q = factorize(n)
print(f"p = {p}")
print(f"q = {q}")
print(f"Diferencia p - q = {p - q}")
print(f"Verificación n = p*q: {p * q == n}")

// Generación de la clave privada

Con p, q y e = 65537 se calcula d = modinv(e, φ(n)) y se exporta la clave en formato PEM:

python3 — generar_clave.py
from Crypto.PublicKey import RSA
from Crypto.Util.Number import inverse

e = 65537
n = p * q
phi = (p - 1) * (q - 1)
d = inverse(e, phi)

key = RSA.construct((n, e, d, p, q))
with open('id_rsa_root', 'wb') as f:
    f.write(key.export_key('PEM'))

print("Clave privada generada: id_rsa_root ✅")
